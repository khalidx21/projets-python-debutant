# 🔄 06 - Vérificateur de palindrome

## 🎯 Objectif

- Manipuler les chaînes de caractères et utiliser les conditions pour vérifier si un mot est un palindrome.

## 📝 Description

Le programme demande à l'utilisateur d'entrer un mot et vérifie si ce mot se lit de la même manière à l'endroit et à l'envers. Il ignore la casse (majuscules/minuscules).

**Exemples :**
- `radar` -> est un palindrome
- `python` -> n'est pas un palindrome

## ✨ Concepts clés

- Slicing de chaînes de caractères (`[::-1]`)
- Méthode `.lower()` pour ignorer la casse
- Conditions `if/else`

## 🚀 Extensions possibles

- Ignorer les espaces et la ponctuation pour vérifier des phrases entières.
- Tester la performance sur de très longues chaînes.
- Créer une fonction réutilisable.
